# Coffee Rating Application

A full-stack Coffee Rating Application where users can vote for their favourite coffee.

## Features
- Displays different coffee items
- Shows coffee price and vote count
- Users can vote for any coffee
- Vote count increases when the Vote button is clicked
- Vote count is saved in MongoDB database
- Vote count remains after page refresh

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Files
- `index.html` – Main webpage
- `style.css` – Design and styling
- `script.js` – Frontend voting functionality
- `server.js` – Backend server and API
- `package.json` – Project dependencies
- `package-lock.json` – Dependency lock file
